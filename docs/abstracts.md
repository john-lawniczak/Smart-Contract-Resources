## Notable Concepts from Whitepapers

### From Bitcoin
- **Proof-of-Work**: Consensus via computational puzzles
- **UTXO Model**: Unspent transaction outputs
- **Blockchain**: Immutable ledger via cryptographic linking

### From Ethereum
- **Smart Contracts**: Self-executing code on blockchain
- **EVM**: Turing-complete virtual machine
- **Gas**: Resource metering mechanism

### From Uniswap V3
- **Concentrated Liquidity**: Capital efficiency via price ranges
- **NFT Positions**: LP positions as non-fungible tokens
- **Multiple Fee Tiers**: Flexible fee structures
