## Getting Started

### For Beginners

1. **Learn Solidity basics**
   - [Solidity by Example](https://solidity-by-example.org/)
   - [CryptoZombies](https://cryptozombies.io/)
   - [Patrick Collins Foundry Course](https://www.youtube.com/watch?v=umepbfKp5rI)

2. **Build projects**
   - Start with simple contracts
   - Contribute to open source
   - Deploy to testnet

3. **Join communities**
   - Discord servers (Foundry, OpenZeppelin, Protocol-specific)
   - Twitter (follow developers and auditors)
   - Ethereum Stack Exchange

### For Security-Focused Careers

1. **Study vulnerabilities**
   - [Damn Vulnerable DeFi](https://www.damnvulnerabledefi.xyz/)
   - [Ethernaut](https://ethernaut.openzeppelin.com/)
   - Read past audit reports

2. **Practice auditing**
   - Review Code4rena contests
   - Write sample reports
   - Learn fuzzing tools (Echidna, Medusa)

3. **Start with bug bounties**
   - Small protocols on Immunefi
   - Code4rena/Sherlock contests
   - Build reputation
